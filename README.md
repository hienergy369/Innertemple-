# InnerTemple

This is the live prototype for InnerTemple, a sacred, psychedelic, interactive temple app.

## How to run locally
1. Open `index.html` in a browser.
2. Interact with the Guided Ritual Journey and Chakra Temple.
3. Chakra orbs play sound and animate mandalas.

## How to host on GitHub Pages
1. Push this repo to a GitHub account.
2. Go to Settings → Pages → Build and deployment → `main` branch → `/ (root)` → Save.
3. Your site will be live at `https://<your-username>.github.io/inner-temple/`.
